1.0 Features and components of GGVTS SrsGGVTS001

This document consist of system requirements for GGVTS (GPS GSM Vehicle Tracking System).

GGVTS/code/gps.h (line 33)

1.1 SrsGGVTS002

Features and components of GGVTS

1.1.1 SrsGGVTS003

Product name GGVTS, should include components GSM, GPS, Processor LPC2148, 2xUART, 1x switch to reset, 1x switch to turn ON/OFF, 1x switch to turn ON PARK MODE, ISP (In system programming).

1.1.2 SrsGGVTS004

GGVTS shall be capable sending SMS when network available.

1.1.3 SrsGGVTS005

GGVTS shall be capable of receiving SMS when network available.

1.1.4 SrsGGVTS006

GGVTS shall be capable of accessing coordinates of host when required and Location is fixed.

1.1.5 SrsGGVTS007

GGVTS shall be capable of sending host location via SMS when asked by user.

1.1.6 SrsGGVTS008

Basic Flow of communication shall have resemblance with this BasicFlowDiagram.

1.1.7 SrsGGVTS009

GGVTS shall be capable of locking host coordinates when PARK MODE is ON.

1.1.8 SrsGGVTS010

GGVTS shall be capable of turning ON (Ignition ON) host when requested by user via SMS.

1.1.9 SrsGGVTS011

GGVTS shall be capable of turning OFF (Ignition OFF) host when requested by user via SMS.

1.1.10 SrsGGVTS012

GGVTS shall contain reset button, on/off button so that in case of urgency GGVTS can be switched ON/OFF or reset.

1.1.11 SrsGGVTS013

GGVTS shall contain 2xUARTS, one to perform live debugging by connecting UART cable to tester and GGVTS and second to do software updates by following ISP button combination.
